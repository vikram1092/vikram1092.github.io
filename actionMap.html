Tools

All tutorials following will require the tools:

  • Macbook

  • Xcode

  • A Web Browser

  • Multimedia access




Tutorials

Tutorial 1: Given access to a Macbook, students will download and set up the integrated development environment (IDE), XCode.

Students will need a Macbook and XCode as basic tools for the course. This tutorial will help them get set up with necessary tools.

Tutorial 2: Given access to the iOS IDE, XCode, students will be able to build and launch a basic application on their device.

Students use XCode to build a basic application. Students also watch a video about how to do this.

Tutorial 3: Given access to XCode materials, students will be able to work with basic views, colors, and images.

Students use XCode to work with application views. Students also watch a video about how to do this.

Tutorial 4: Given access to XCode materials, students will be able to design table views in their apps.

Students use XCode to work with table views.

Tutorial 5: Given access to XCode materials, students will be able to design tab bar views in their apps.

Students use XCode to work with tab bar views. Students also watch a video about how to do this.

Tutorial 5: Tutorial 6: Given access to XCode materials, students will be able to design navigation view controllers in their apps.

Students use XCode to work with navigation views.

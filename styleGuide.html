Style Guide

Technical Standards


Web Safe Typography Palette

The tutorial site will use a Google Font, Source Sans Pro (sans serif), for all categories of font, including headings, body, and image font. If this is not available, the website will automatically use Helvetica Nueue (Mac) or Arial (universal) for all its purposes. This is done to keep the experience consistent across sections, headings, and elements on the webpage.
The difference will be in the font weight. Headings will be large, all caps, and use a bold font type. Image font will be regular sized yet bold. Body text will generally use default weighted font with italics and bold used to define elements and emphasis as needed.

File Naming Conventions


File names will consist of lowercase letters, and uppercase letters.
File names should not include numbers that reference location or sequence.
File names must start with “tutorial”.
File names must use capitalization to indicate the start of every new word (eg: “tutorialHome”).
File names shall be descriptive of the file.


Multimedia


Images will all contain alt tags that provide a description of the item.



Logical Standards


Content Color Behavior

Background colors must be the theme blue or white.
Text over the blue color must be white.
Text over white backgrounds must be the faded black color from the theme.
Hover highlight colors used must always be the dark blue theme color.
Headers must be supplemented by a border element underneath to distinguish it from the body.


Content Layout


All content must allow ample padding to emphasize space and simplicity.
All headings and images must be centered in the page.


Page Layout


All tutorials must be available and described on the home page.
A navigation bar must allow free navigation regardless of the page user is on.



Pedagogical Standards


Interface

Students must be able to navigate to any course they need to, but lessons must encourage a linear flow from the end of one to the beginning of the next.


Lessons


Lessons must incorporate screenshots and multimedia where complicated matter is being taught.

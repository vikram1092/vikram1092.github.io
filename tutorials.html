iOS For Beginners


Formative Assessment

Tutorials


Outcomes

Given access to this lesson, students will be able to set up the necessary environment for iOS programming by downloading the necessary tools and registering for the Apple Developer Program.


Modules


  1. Intro: Given access to a Macbook, students will download and set up the integrated development environment (IDE), XCode.

  2. XCode, Basic App Structure, And Testing: Given access to the iOS IDE, XCode, students will be able to build and launch a basic application on their device.

  3. Basic Views: Given access to XCode materials, students will be able to work with basic views, colors, and images.

  4. Table Views : Given access to XCode materials, students will be able to design table views in their apps.

  5. Tab Bar Views: Given access to XCode materials, students will be able to design tab bar views in their apps.

  6. Navigation Views: Given access to XCode materials, students will be able to design navigation view controllers in their apps.
